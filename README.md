# AnimeAI ✨

**比 DeepSeek 官方客户端更强大的二次元 AI 聊天 App**

支持 OpenAI / Anthropic / Gemini 三种兼容格式，内置 5 种 AI 人格、30+ 模型 SVG 图标、13 项创新功能。

## 特性

- 🌸 **粉色二次元风格** — 毛玻璃全面屏沉浸式 UI
- 🎭 **5 种内置人格** — 3 个二次元 + 助手 + 开发工程师
- 🧠 **推理模型支持** — 自定义思考强度
- 🖼️ **图片生成 & 视频生成**
- 📝 **完整 Markdown 渲染**
- 🌐 **多接口兼容** — OpenAI / Anthropic / Gemini
- ⚙️ **完全自定义** — Base URL / API Key / Model ID

### 创新功能

| 功能 | 说明 |
|------|------|
| AI 人格竞技场 | 多 AI 同台 PK，投票对比 |
| 跨模型交叉验证 | 同一个问题多个模型回答对比 |
| 对话时间胶囊 | 封存对话，未来解锁 |
| 对话树/平行宇宙 | 分支回溯，树状可视化 |
| 代码沙箱 | 内嵌 Python/JS 执行环境 |
| AI 工作流 | 可视化拖拽节点流水线 |
| 本地知识库 RAG | 导入 PDF/TXT 智能检索 |
| 智能仪表盘 | 实时 Token / 情绪曲线 |
| 自定义人格训练场 | 性格滑块 + 自动生成 Prompt |
| 睡眠模式 | AI 后台自对话 |
| 截图分享卡片 | 6 种风格精美卡片 |
| Prompt 魔导书 | 模板市场一键使用 |
